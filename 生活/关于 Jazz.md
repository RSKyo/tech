## 1. Soul Jazz（灵魂爵士）

**核心来源**

- 1950–60 年代美国
    
- Jazz × Blues × Gospel × R&B
    

**音乐特征**

- 节奏：中速偏慢，强烈律动感（groove）
    
- 和声：相对简单、偏布鲁斯
    
- 乐器：风琴（Hammond B3）、钢琴、萨克斯、鼓
    
- 气质：**温暖、有人味、略带宗教感**
    

**情绪关键词**

> 温厚 / 真实 / 人情味 / 城市夜晚

**常见场景**

- 老派酒吧
    
- 黑胶唱片
    
- 深夜聆听
    

---

## 2. Smooth Jazz（柔滑爵士）

**核心来源**

- 1970–90 年代
    
- Jazz × Pop × Soft R&B
    

**音乐特征**

- 节奏：平稳、干净、不过分复杂
    
- 和声：悦耳、流畅，旋律导向
    
- 乐器：萨克斯、电子钢琴、合成器
    
- 制作：高度“抛光”，商业友好
    

**情绪关键词**

> 放松 / 流畅 / 轻奢 / 无压力

**常见场景**

- 咖啡馆
    
- 商场背景音乐
    
- 驾车、办公
    

---

## 3. Cozy Jazz（舒适爵士）※偏现代分类

**核心概念**

- **情绪型标签**，不是严格历史流派
    

**音乐特征**

- 节奏：慢或中慢
    
- 动态：小音量、低对比
    
- 乐器：钢琴、低音提琴、刷鼓
    
- 氛围：亲密、包裹感强
    

**情绪关键词**

> 温馨 / 安静 / 室内 / 雨天

**常见场景**

- 夜晚学习
    
- 阅读
    
- 独处
    

---

## 4. Groovy Soul Jazz（律动型灵魂爵士）

**核心来源**

- Soul Jazz 的**律动强化版本**
    

**音乐特征**

- 节奏：明显 groove，鼓和贝斯突出
    
- 律动：可点头、可摇摆
    
- 风格：更偏 Funk / R&B
    
- 情绪：更外放、更“动”
    

**情绪关键词**

> 摇摆 / 活力 / 街头感 / 自信

**常见场景**

- 白天播放
    
- 轻运动
    
- 创意工作
    

---

## 5. Uplifting Smooth Jazz（积极向上的柔滑爵士）

**核心概念**

- Smooth Jazz 的**情绪升级版**
    

**音乐特征**

- 节奏：中速偏快
    
- 调式：多用大调
    
- 旋律：明亮、上扬
    
- 情绪：不激烈，但正向
    

**情绪关键词**

> 阳光 / 希望 / 轻快 / 正能量

**常见场景**

- 上午
    
- 办公室
    
- 心情提振但不吵
    

---

## 6. Chill Hip-hop Jazz（冷静嘻哈爵士）

**核心来源**

- Jazz × Hip-hop × Lo-fi
    
- 90s Jazz Rap / Beat Culture 延伸
    

**音乐特征**

- 节奏：Hip-hop 鼓型（boom bap / lo-fi）
    
- 采样：爵士钢琴、萨克斯、老唱片
    
- 气氛：慵懒、低饱和
    
- 常见：无歌词或低存在感人声
    

**情绪关键词**

> 慵懒 / 冷静 / 夜晚 / 都市感

**常见场景**

- 学习
    
- 编程（这一点你可能会有共鸣）
    
- 深夜独处

## 1. Deep Soul Jazz（深度灵魂爵士）

**定位**

- Soul Jazz 的**情绪深化版本**
    
- 重点不在“动”，而在“沉”
    

**音乐特征**

- 节奏：慢～中慢，律动被弱化
    
- 和声：偏小调，延展音多，情绪张力强
    
- 演奏：更内省，留白多
    
- 音色：厚重、暗色调
    

**情绪关键词**

> 沉思 / 深夜 / 情绪重量 / 内心独白

**使用场景**

- 夜深独处
    
- 情绪整理
    
- 长时间专注（不适合提神）
    

**一句话理解**

> Deep Soul Jazz 是“**往内走的 Soul Jazz**”。

---

## 2. Funk Soul Jazz（放克灵魂爵士）

**定位**

- Soul Jazz + Funk 的**节奏主导型变体**
    
- Groove 是第一位
    

**音乐特征**

- 节奏：强烈、切分明显
    
- 贝斯：线条感极强，常为主角
    
- 鼓：Funk 鼓组，律动清晰
    
- 情绪：外放、身体驱动
    

**情绪关键词**

> 摇摆 / 街头 / 能量 / 身体感

**使用场景**

- 白天
    
- 需要节奏驱动的工作
    
- 轻运动、行走
    

**一句话理解**

> Funk Soul Jazz 是“**让身体先于大脑反应的爵士**”。

---

## 3. Hush Jazz（低语爵士 / 静谧爵士）

**定位**

- **氛围型、音量极低的爵士**
    
- 更多是播放语境，而非传统流派
    

**音乐特征**

- 音量：极低
    
- 动态：几乎无起伏
    
- 乐器：轻钢琴、刷鼓、柔和低音
    
- 结构：简单，重复性强
    

**情绪关键词**

> 私密 / 安静 / 克制 / 贴近

**使用场景**

- 深夜
    
- 阅读、冥想
    
- 需要“存在但不被注意”的音乐
    

**一句话理解**

> Hush Jazz 是“**会自动后退一步的爵士**”。

---

## 4. Modern Jazz（现代爵士）

**定位**

- **时间概念**而非情绪概念
    
- 泛指 1990s 至今的当代爵士创作
    

**音乐特征**

- 风格：高度多样（融合电子、世界音乐、实验）
    
- 节奏：可能复杂、也可能极简
    
- 和声：更自由，不追求传统 Swing
    
- 制作：现代录音、空间感强
    

**情绪关键词**

> 前沿 / 探索 / 多元 / 不被定义

**使用场景**

- 主动聆听
    
- 对音乐结构有兴趣的人
    
- 创作灵感获取
    

**一句话理解**

> Modern Jazz 是“**不再被‘爵士该怎么写’束缚的爵士**”。

---

## 5. Noir Jazz（黑色爵士 / 黑色电影爵士）

**定位**

- 强烈视觉与叙事导向
    
- 源自 Film Noir（黑色电影）审美
    

**音乐特征**

- 调式：多用小调
    
- 节奏：慢、中慢，紧张但克制
    
- 乐器：萨克斯、低音提琴、暗色钢琴
    
- 氛围：阴影感、故事感强
    

**情绪关键词**

> 冷峻 / 悬疑 / 孤独 / 城市夜色

**使用场景**

- 夜晚
    
- 写作、构思故事
    
- 城市雨夜、独处
    

**一句话理解**

> Noir Jazz 是“**带剧情的爵士**”。
